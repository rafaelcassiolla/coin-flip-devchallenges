# 🪙 Coin Flip - devchallenges

Uma aplicação web simples e interativa que permite aos usuários lançarem uma moeda virtual para obter "Cara" ou "Coroa" de forma totalmente aleatória. Este projeto foi desenvolvido como parte dos desafios da plataforma **devchallenges**.

[Acessar o Projeto Demonstrativo](https://seu-usuario.github.io/seu-repositorio)

---

## 🎯 Objetivos de Aprendizagem

Este projeto teve como foco solidificar os fundamentos do desenvolvimento web front-end, permitindo praticar:
* **Sintaxe Essencial:** Compreensão prática de HTML5, CSS3 e JavaScript Moderno.
* **Manipulação do DOM:** Captura de elementos e atualização dinâmica de conteúdo na página.
* **Tratamento de Eventos:** Escuta e resposta a interações do usuário (como cliques em botões).
* **Design Responsivo:** Criação de um layout atraente que se adapta a diferentes tamanhos de tela (computadores, tablets e celulares).

---

## ⚙️ Requisitos e Funcionalidades

* **Lançamento Virtual:** Botão "Jogar Moeda" que aciona a lógica do sorteio.
* **Aleatoriedade Real:** Uso de funções matemáticas para garantir 50% de chance para cada resultado.
* **Feedback Visual:** Exibição imediata do resultado ("Cara" ou "Coroa") na tela.
* **Estilização Customizada:** Uso de bordas e sombras marcantes baseadas no design proposto.
